# NaraEdu: Student Management System with AI Integration 🚀

[![Framework](https://img.shields.io/badge/.NET-8.0-blueviolet)](https://dotnet.microsoft.com/)
[![Database](https://img.shields.io/badge/Database-SQL--Server-blue)](https://www.microsoft.com/en-us/sql-server/)
[![AI](https://img.shields.io/badge/AI-Ollama--Nara-orange)](https://ollama.com/)

**NaraEdu** adalah aplikasi desktop modern untuk manajemen data siswa (CRUD) yang dikembangkan dengan C# WinForms. Aplikasi ini tidak hanya menyimpan data, tetapi juga dilengkapi dengan asisten cerdas bernama **Nara** yang terintegrasi langsung melalui Ollama.

---

## ✨ Fitur Unggulan

* **⚡ Modern CRUD Operations**: Manajemen data siswa (NISN, Nama, Alamat, dll) dengan validasi data yang ketat.
* **🤖 Nara AI Assistant**: Asisten virtual yang tertanam di dalam aplikasi untuk membantu interaksi pengguna dan menjawab pertanyaan secara real-time.
* **📂 Entity Framework Core**: Menggunakan ORM modern untuk interaksi database yang lebih aman, cepat, dan terstruktur dibandingkan SQL manual.
* **🎨 Guna2 UI Framework**: Tampilan antarmuka "Dark Mode" yang estetis, responsif, dan nyaman di mata (VS Code Inspired).
* **🔍 Advanced Search**: Pencarian data siswa yang cepat dan efisien.

---
